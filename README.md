# KMR Invoice Billing

This repository is focused on the invoice billing application.

## Billing App Location

The active app is inside `website/`.

## Run Locally

**Prerequisites:** Node.js

1. Open terminal in `website/`
2. Install dependencies:
   `npm install`
3. Run the billing server:
   `npm run dev`
4. Open:
   `http://localhost:4010`
